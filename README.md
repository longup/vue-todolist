# vue-todolist

> A Vue.js project


## Build Setup

``` bash
# install dependencies(安装依赖)
npm install

# serve with hot reload at localhost:8080（开发环境下运行）
npm run dev

# build for production with minification（生成环境下运行）
npm run build
```
